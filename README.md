# 1Luisin - Landing Page

Landing page pessoal feita em HTML e CSS para apresentar perfil, sobre, projetos e contato.

## Demo local

1. Abra o arquivo `index.html` no navegador.
2. Pronto: a pagina carrega os estilos e imagens via `assets/`.

## Estrutura do projeto

- `index.html`: markup principal da landing page.
- `assets/`: estilos, imagens e recursos visuais.
  - `assets/css/`: folhas de estilo separadas por area.
  - `assets/img/`: imagens e icones usados na pagina.

## Funcionalidades

- Layout responsivo com secoes de apresentacao, sobre, projetos e contato.
- Menu de navegacao no topo.
- Area de contato com formulario (front-end).
- Links para redes (GitHub/GitLab/LinkedIn).

## Tecnologias

- HTML5
- CSS3

## Personalizacao rapida

- Textos: edite `index.html`.
- Cores e layout: edite os arquivos em `assets/css/`.
- Imagens: substitua os arquivos em `assets/img/`.
